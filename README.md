# 毕业设计-社团信息管理系统

​	这是我的毕业设计

## 如何部署？

使用[小皮面板](https://www.xp.cn/)

将整个项目复制到服务器的文件夹下

“软件管理”中下载PHPMyAdmin

之后会到软件的“首页”，点击“数据库工具”旁的打开，点击PHPMyAdmin

输入你数据库的用户名和密码，进入数据库（默认是root和root）

进入数据库后点击“导入”，选择下载的“is.sql”，按照步骤导入

不要尝试把这个系统部署在公网上，这只是个简单的系统，并没有非常高的安全系数
